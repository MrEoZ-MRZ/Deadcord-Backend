Congrats, you found the help menu. Here you can find what the different buttons do and what parameters they require.

Deadcord Features

  • 📡 Ping Tokens. When clicked, this will ping all of your tokens and move invalid or phone locked tokens to a seperate file.
  • ✏️ Change Bot Nicknames. When clicked, it will change all current bot nicknames in the focused server. Message bar parameters are shown below.
    • <nickname/random> string
  • 👁️ Disguise Bots. Randomly change the bots nickname and profile picture.
  • ✉️ Speak in all channels. Send a singular message in all open channels in a server.
  • 😀 Spam Reactions. Spam reactions for the specified message.
    <channel_id>,<message_id>,<emoji_name> (smile)
  • ⬆️ Join Server. Join the specified server safely. Join times are randomized.
    <invite_url>
  • ⬇️ Leave Server. All bots leave the specifid server.
  • ✅ Start Spam. When clicked, all bots start spam in the specified server.
    <messages_seperated_by_comma>
  • ❌ Stop Spam. When clicked, all spam from bots will stop until the start spam button is clicked again.
